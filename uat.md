
# UAT (Unity application template)

+ Sources - Основные ресурсы
    + Primitives - Элементарные данные 
        + Images
            + Textures
            + Maps
            + Icons
            + Atlases
            + ...
    
        + Audio
            + Music
            + Sounds
            + ...

        + Meshes
            + Props
            + Characters
            + Buildings
            + ...

        + Fonts
        + Video
        + Text
        + Materials
        + ...

    + Complex - Элементарные данные, сгруппированные в законченные сущности
        + Models
            + Vehicles
            + Characters
            + Buildings
            + ...
        + Controls
        + Forms
        + Screens
        + ...

    + Containers - Сущности, объединяющие элементарные и сгруппированные данные
        + Prefabs
            + Terrain
        + Scenes
        + Bundles
        + Configs
        + ...

+ Scripts - Код
    + Bases
        + Abstracts
        + Headers
        + Enums

    + Structures
        + DataTypes
        + Entities
        + Commands
        + Handlers
        + States
        
    + Managers
    + Controllers
    + Helpers
        + Converters
        + Adapters
        + Checkers
        + Loaders
        + Factories
        + Extensions
    + Shaders

+ Extras - Сторонние дополнения, расширения, плагины, системы и т.д.
    + Plugins
    + Packages
    + ...

+ Sandbox - Временные и сгенерированные данные
    + Temp
    + Generated
    + Experimental
    + ...
